# 04 — BUILDING A FORTRESS WHEN A FENCE LINE WILL DO

**Point 4: DVS is intentionally minimal**
**Framework:** Structural Overkill

**The Logic:** Other skill systems build custom sandboxes, kernel-level isolation, VM containers, and policy engines to keep skills contained. DVS uses the URL boundary as the security fence — the same principle as the browser same-origin policy, one of the most successful security boundaries in software history. Same-origin isolation falls out naturally from URL scoping: stay within your Trust Root prefix. No new code, no custom runtime, no policy language.
