## Short Tweet

We just submitted "Domain-Verified Skills (DVS) Protocol" as an IETF Internet-Draft (draft-zzn-dvs-00).

DVS lets AI agents discover and execute skill definitions served over HTTPS, using domain-based trust instead of centralized registries.

Read it: https://datatracker.ietf.org/doc/draft-zzn-dvs/

Feedback welcome — especially from folks working on AI agent interop, web discovery, and trust models. What are we missing?
