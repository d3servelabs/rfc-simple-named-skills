# 04 — TRUSTING A NAME TAG AT A COSTUME PARTY

**Point 2: Current trust is vibes-based**
**Framework:** Signal vs. Noise

**The Logic:** Today's skill ecosystem works on vibes — the name looks right, the description sounds legitimate, you install it. This is the same model that gave us typosquatting on npm, malicious browser extensions, and fake mobile apps.
