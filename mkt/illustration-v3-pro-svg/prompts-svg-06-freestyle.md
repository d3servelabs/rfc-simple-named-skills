# 06 — THE GAP IS THE RISK

Create an animated SVG infographic showing a dangerous widening gap between two curves.

One curve is agent capabilities — growing exponentially: text generation, code execution, tool use (MCP), computer use, multi-agent delegation (A2A), autonomous coding, full autonomy. A hockey stick shooting upward.

The other curve is trust verification at the skill layer — barely moving, a flat line near zero.

The widening gap between the two curves IS the risk. Shade it red. Every increment of autonomy without verification is unpriced liability. Show that DVS closes this gap.
